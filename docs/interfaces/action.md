# Interface: Action<T\>

Represents an action on an item of type `T`.<br>
  Example: ``const logUserAction: Action<User> = user => console.log(`User ${user.name} (id: ${user.id})`);``

## Type parameters

Name | Description |
:------ | :------ |
`T` | The type of the item the action is defined on.    |
